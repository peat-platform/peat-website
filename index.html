Welcome to PEAT

PEAT stands for Personal Data, Apis, and Trust. PEAT is an open source project that aspires to put end users back in control of their data by providing a user-cenric cloud based data storage platform. The PEAT platform maximises user control by providing an OAuth compliant secure fine-grained access control mechanism, through which, users can control access to their data. Additionally, PEAT aims to increase the end-user's awareness of how their data is accessed through its innovative user-dashboards. PEAT also simplifies integration with 3rd party cloud based services through its API aggregator, its client side library, and service enablers; all of which allow application development in a fraction of the time.

Personal Information

PEAT allow your users manage their personal information across many mobile applications and devices.

API's

PEAT's advanced REST API framework connects your users to a multitude of cloud-based services.

Trust

PEAT allows your users to safeguard their data via a standards based compliant security framework.

PEAT's Philosophy

Personal Data

PEAT stores user data in Personal Cloudlets. The Cloudlet is built up from a user interataction with OPENi enabled apps. It is an easy to use free storage facility that affords them full control over the storage and sharing of their personal data.

API's

APIs are what connect us, but they can be difficult to manage. In PEAT we abstract this inherent volitility in dealing with cloud based services by providing our own simple community managed Graph API, this greatly increases the power and reach of PEAT enabled applications.

Trust

PEAT aims to inspire innovation in the European mobile applications industry, by radically improving the trust in personal cloud storage through the development of a consumer-centric, open source mobile cloud applications platform.

Mobile Client Libraries

In line with PEAT's ethos of openness and collabotation we have included a number of client libraries which greatly reduce the developement overhead of PEAT enabled applications.
