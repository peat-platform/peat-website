Welcome to PEAT

PEAT stands for Personal Data, Apis, and Trust, PEAT is an opensource project that aspires to put end users back in control of their data by provided a user-cenric cloud based data storage platform. The PEAT platform maximises user control by providing an OAuth compliant secure fine-grained access control mechanism through which user can control access to their data. Additionally, PEAT aim to increase the end-user's awareness of how their data is access through its innovative user-dashboards. PEAT also happens to simplify integration with 3rd party cloud based services through its API aggregator, its client side library, and service enabler; all of which allow application development in a fraction of the time.

Personal Information

PEAT allow your users manage their personal information across many mobile applications and devices.

API's

PEAT's advanced REST API framework connects your users to a multitude of cloud-based services.

Trust

PEAT allows your users control their data safeguarded by a standards based compliant security framework.
